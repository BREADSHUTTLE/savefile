* AnswerLobby.cs : 서버에서 lobby 정보 api로 받아 올 때 포맷 (보통의 컨텐츠 포맷)

* RequestLobby.cs : 서버에 lobby 정보 api 요청 할 때 포맷 (보통의 컨텐츠 포맷)

* Session.cs : SessionLobby와 SessionGame으로 구분 될 수 없는 메인 Session

* SessionGame.cs : 인게임 내 통신 공통 기능

* SessionLobby.cs : 아웃게임 내 통신 공통 기능

* AppsFlyerSystem.cs : 앱스플라이어 연동 이후 관련 작업

* ClientInfo.cs : 클라이언트 정보

* CommonTools.cs : 게임 내 공통으로 사용 해야하는 각종 기능

* FacebookFeedInfo.cs : 피드 관련 data pool

* Lobby.cs : 로비 메인 화면 구성

* LobbyInfo.cs : 로비 구성 data pool

* MissionInfo.cs : 미션 정보 data pool

* PopupMission.cs : 미션 컨텐츠 팝업 구성

* MissionItem.cs : 미션 컨텐츠 팝업 내 UI 일부 기능

* PopupAttendance.cs : 출석 컨텐츠 팝업 구성

* PopupStore.cs : 상점 컨텐츠 팝업 구성

* StoreListView.cs : 상점 컨텐츠 내 리스트 뷰 (공통 포맷)

* StoreListViewItem.cs : 상점 컨텐츠 내 리스트 뷰 구성 UI

* TimerSystem.cs : 클라이언트 접속 시, 필요한 타이머 기능

* TopUI.cs : 인/아웃 공통 상단 UI 기능

* UICanvasListView.cs : UGUI 리스트뷰 커스텀 기능

* UIGradationColor.cs : 2D 텍스트 및 이미지 그라데이션 효과 넣는 커스텀 기능

* WorldMessage.cs : 게임에서 보여질 월드메시지 기능
